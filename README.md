## ‚ú® Features

- **Anonymous Posting:** No accounts, no profiles. Just your words.
- **Real-time Feed:** Messages are fetched instantly from a Supabase backend.
- **Aesthetic UI:** A typewriter-inspired, sticky-note grid layout.

- **Mobile Responsive:** Works perfectly on phones, tablets, and desktops.
